
# 🏰 ВПИ Telegram Бот (Военно-Политическая Игра) — Версия 2.0

Полностью асинхронный Telegram-бот на базе **Aiogram 3** и **SQLite**, разработанный для автоматизации текстовых Военно-Политических Игр (ВПИ). Бот позволяет игрокам регистрировать свои государства, управлять экономикой, создавать геополитические альянсы и проводить секретные спецоперации против соседей.

## 📁 Структура проекта

```text
├── bot.py               # Главный исполняемый файл (Точка входа, Polling)
├── config.py            # Модуль конфигурации и безопасной загрузки секретов
├── database.py          # Инициализация SQLite и методы работы с БД
├── handlers.py          # Игровые хендлеры, FSM-состояния и бизнес-логика
├── requirements.txt     # Список зависимостей проекта
└── .env                 # Файл с приватными ключами (Игнорируется Git)
```

## 🛠 Инструкция по установке и запуску

### 1. Клонирование репозитория
```bash
git clone [https://github.com/ВАШ_НИК/ВАШ_РЕПОЗИТОРИЙ.git](https://github.com/ВАШ_НИК/ВАШ_РЕПОЗИТОРИЙ.git)
cd ВАШ_РЕПОЗИТОРИЙ
```

### 2. Настройка виртуального окружения
Рекомендуется использовать изолированное окружение, чтобы не засорять систему:
```bash
# Для Linux/macOS
python3 -m venv venv
source venv/bin/activate

# Для Windows
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения
В корневой папке проекта создайте файл `.env` и впишите туда токен вашего бота, полученный от @BotFather:
```env
BOT_TOKEN=1234567890:ABCdefGhIJKlmNoPQRsTUVwxyZ
```
> **Важно:** Никогда не добавляйте файл `.env` в коммиты на GitHub! На реальном хостинге (например, Bothost, Fps.ms или Render) этот токен нужно будет вписать в раздел **Variables / Environment Variables** (Переменные окружения) в панели управления.

### 5. Запуск бота
При первом старте бот автоматически создаст локальный файл базы данных `vpi_game.db` и развернет все необходимые таблицы:
```bash
python bot.py
```

## 🎮 Доступные игровые команды

* `/start` — Регистрация новой державы.
* `/me` — Просмотр карточки своего государства и секретных складов ВПК.
* `/spy @username` — Платный шпионаж за доходами и армией другого игрока.
* `/sabotage @username` — Попытка устроить диверсию (шанс успеха 50% на основе кубика).
* `/create_union` — Интерактивный конструктор создания альянса с загрузкой флага.
* `/list_unions` — Список существующих геополитических блоков.
* `/union [ID]` — Просмотр досье альянса и его официального флага.
* `/join_union [ID]` — Вступить в выбранный альянс.
